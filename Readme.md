# IoT based Water quality monitoring system

## Features
- Inbuilt display
- Temperature monitoring
- Turbidity monitoring
- pH monitoring
- Android app support
- ThingSpeak server support


## Components
- Atmega328p
- Turbidity sensor
- pH Sensor
- DS18B20 temperature sensor
- 16x2 LCD Display
- ESP8266
- 10K pot
- 16MHz Crystal
- 7805 Voltage regulator
- some capacitors and resistors
